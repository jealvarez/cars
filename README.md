# **Cars**

## **Description**

The Cars application provides full control over any cars inventory.

## **Run Locally**

1. Open a terminal
```ssh
$ make dotnet-develop
```

2. Navigate to [http://127.0.0.1:8000](http://127.0.0.1:8000)

## **Run with Docker**

1. Open a terminal
```ssh
$ make docker-build tag=v1.0
$ make docker-run tag=v1.0
```

2. Navigate to [http://127.0.0.1:8000](http://127.0.0.1:8000)
